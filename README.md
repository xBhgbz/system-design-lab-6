# Hotel Booking API - Event-Driven Architecture

Лабораторная работа №6: Проектирование Event-Driven архитектуры

### Выполнил: Кудрин Ярослав Владимирович, М8О-102СВ-25, вариант 13

REST API сервис для бронирования отелей с применением RabbitMQ. Взял за основу сервис из 2 лабораторной и доделал в нем взаимодействие через брокер сообщений.

## Функционал
- Регистрация и аутентификация пользователей (Session Token).
- Управление отелями (создание, поиск).
- Управление бронированиями (создание, просмотр, отмена).

## Запуск через Docker

```bash
docker-compose up --build
```
